\# WFSL Proof Surfaces



This document links to verifiable proof of execution and composition.



\## Continuous Integration



\- wfsl-shell-guard  

&nbsp; GitHub Actions runs verify execution preflight.



\- wfsl-evidence-guard  

&nbsp; GitHub Actions enforce Shell Guard dependency and emit evidence.



\- wfsl-admission-guard  

&nbsp; GitHub Actions enforce Shell Guard before surface admission.



\- wfsl-governance-chain  

&nbsp; GitHub Actions orchestrate and verify the full enforcement chain.



Each proof is visible under the Actions tab of the respective repository.
