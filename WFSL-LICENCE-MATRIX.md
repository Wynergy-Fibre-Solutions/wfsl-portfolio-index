\# WFSL Licence Matrix



This document defines the licence classes used across the WFSL portfolio.

Licence is enforced in execution logic, not by interpretation.



---



\## Licence Classes



\### WFSL Open Core

\- Public audit and inspection

\- Forking permitted

\- No commercial use

\- No resale or hosted service



\### WFSL Community / CIC

\- Internal deployment by charities and CICs

\- Modification permitted

\- Non-commercial use only



\### WFSL Demo / Evaluation

\- Time-limited execution

\- Non-production

\- Watermarked and traceable



\### WFSL Commercial

\- Production use

\- Licence-bound execution

\- Cryptographic admission required



\### WFSL Internal

\- WFSL-only systems

\- Not distributable



---



\## Enforcement



All WFSL software emits evidence binding:

\- Component identity

\- Execution context

\- Licence class



Unauthorised execution is detectable by design.
